# Security Policy
### Hermes Sovereign Runtime · Cyber Strategy Institute

## Supported Versions

| Version | Supported |
|---|---|
| 1.x (current) | ✅ Active |

## Reporting a Vulnerability

**Email:** security@cyberstrategyinstitute.com

**PGP:** Available on request.

**Do not** open a public GitHub issue for security vulnerabilities.

**Response timeline:**
- Acknowledgment: 48 hours
- Assessment: 5 business days
- Coordinated disclosure: within 90 days of report

## Scope

In scope for this package:
- AI SAFE² gateway (`gateway/gateway.py`, `gateway/scanner.py`)
- Memory vaccine effectiveness bypass
- Kill switch circumvention
- Audit log integrity bypass
- Docker Compose stack escape

Out of scope:
- Hermes Agent core (report to NousResearch)
- HashiCorp Vault (report to HashiCorp)

## Credits

We publicly acknowledge security researchers who responsibly disclose valid findings.
