Do / Don't Quick Guide

✅ DO

Use FORGE Fire (#E8600A) as primary action color. Use Bebas Neue for all display text. Keep tagline in full: "Every Dollar to Citizens. Every Tool Open Source. Every Community Defended." Always include forge.gov. Use #FORGEAct on every post.

❌ DON'T

Don't use purple gradients or generic tech aesthetics. Don't change "not one cent to mega-corps" — this line is non-negotiable. Don't remove open-source requirement language. Don't use military acronyms in public-facing content.
