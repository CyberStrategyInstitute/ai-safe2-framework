## 📋 What Is the FORGE Act?

The FORGE Act is a citizen-first AI funding framework that:

- Creates **28–2,873 funded AI opportunities per congressional district** (depending on funding tier: $5B–$500B)
- Directs **zero dollars to mega-corporations** — only to individuals and small teams (under 50 employees)
- Requires **all funded innovations to be open source** under OSI-approved copyleft licenses
- Funds **distributed cyber defense** across all 3,143 US counties
- Is **self-sustaining by Year 5** through an Ecosystem Reinvestment Fund (ROI recycling)
- Establishes a **Solution RFI process** — submit a working GitHub repo, win a federal contract in 90 days

The full policy brief is available at: [`FORGE_Act_American_Marshall_Plan_for_AI_v3.pdf`](../FORGE_Act_American_Marshall_Plan_for_AI_v3.pdf)  
The marketing strategy document is available at: [`FORGE_Act_Marketing_Plan.pdf`](../FORGE_Act_Marketing_Plan.pdf)

## 🗣️ Sharing These With People You Know

You know your people better than any campaign does. These materials are starting points — adapt the message to fit how you actually talk.

If someone in your life would connect more with one framing than another, here's a simple way to think about it:

- **Someone who cares about American legacy and institutional strength** → the Marshall Plan framing resonates — it's about citizen investment, not corporate welfare, and it echoes programs that worked
- **Someone skeptical of everything, especially government and corporations** → the build-it-yourself framing meets them where they are — it validates the skepticism and offers something concrete to do with it
- **Someone driven by purpose and frustrated by systems that didn't deliver** → the real-impact framing speaks to wanting genuine accountability and outcomes that are measurable
- **Someone who grew up online and fact-checks everything in real time** → lead with the documented evidence — they respect receipts, not sales pitches
- **A parent worried about what their kids are inheriting** → the parenting framing makes the stakes personal without being alarmist

The FORGE Act is about making a future free from control. It's also about building community, meeting people where they already are and connecting something real — the FORGE Act — to the things they already care about. The message is the same. The door you may walk through can look different, but you aren't alone.

## 🌐 Viewing Files as Web Pages (GitHub Pages)

By default, GitHub shows HTML source code rather than rendering the files. To share these as live, clickable web pages:

1. Go to your repo **Settings → Pages** (left sidebar)
2. Under "Source," select **Deploy from a branch**
3. Choose `main` branch, `/ (root)` folder → **Save**
4. Wait 2–3 minutes, then your files will be live at:

```
https://cyberstrategyinstitute.github.io/ai-safe2-framework/FORGE-Act/FORGE-Act_Overview.html
```

Each file gets its own shareable URL — clean enough for a social media post, a QR code on a printed flyer, or a link in an email to your representative's office.

## 🔧 Adapting These Files

All files are self-contained HTML with embedded CSS — no build tools, no dependencies, no npm install. To edit:

1. Open any `.html` file in any text editor (VS Code, Notepad, TextEdit all work)
2. Find the content sections — they're clearly commented in the code
3. Update district-specific numbers, local names, or personal stories
4. Save and print, or commit back to the repo

To change the color palette, edit the `:root` CSS variables at the top of each file:

```css
:root {
  --fire: #E8600A;   /* primary orange — CTAs, headlines */
  --navy: #0B1D3A;   /* deep navy — trust, authority */
  --gold: #FFD60A;   /* gold — accents, stats */
}
```
